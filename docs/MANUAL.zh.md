# AI Assistant — 用户操作手册

## 这是什么？

AI Assistant 是一个智能聊天工具，它可以连接各种 AI 模型。除了聊天，它还可以通过插件控制 **Blender**（3D 建模软件）和 **GIMP**（图像编辑软件）—— 你告诉 AI 想做什么，AI 帮你操作软件。

---

## 1. 聊天 — 和 AI 对话

这是主界面，你在这里和 AI 聊天。

![聊天主界面](img/01_chat_overview.png)

### 左侧 — 会话列表

你所有的对话都列在这里。每个会话独立保存聊天记录。

| 按钮 | 作用 |
|---|---|
| **+ New** | 新建一个会话 |
| **铅笔图标** | 重命名会话 |
| **垃圾桶图标** | 删除会话 |

### 中间 — 聊天区域

- 你的消息显示在 **右侧**（蓝色气泡）
- AI 回复显示在 **左侧**（灰色气泡）
- AI 回复会实时逐字显示（流式输出）

### 底部工具栏

![选择模型和角色](img/05_chat_selectors.png)

- **左边下拉框**：选择用哪个 AI 模型（如 DeepSeek、GPT-4o、Claude）
- **右边下拉框**：选择"角色"——告诉 AI 用什么样的风格回答（如"3D 设计师"、"图片编辑师"）
- **Token 计数**：显示当前会话用了多少 Token

### 怎么用

1. 点击 **+ New** 开始一个新的会话
2. 在底部输入框输入你想说的话
3. 点击 **发送** 按钮或按回车键
4. 等待 AI 回复

![输入消息](img/03_chat_typing.png)

> **小提示**：鼠标悬停在你自己的消息上，可以看到"复制"和"重新输入"按钮。

## 2. AI 桥接 — 连接 Blender / GIMP

这个页面管理 AI 和外部工具的连接。连上之后，AI 就能根据你的指令操作 Blender 或 GIMP。

![AI 桥接概览](img/07_mcp_overview.png)

### 界面说明

- **标签栏**：显示所有可用的服务（Blender、GIMP 等）
- **状态徽章**：绿色 = 已连接，灰色 = 未连接
- **日志区域**：实时显示活动日志（每 3 秒自动刷新）

![服务详情](img/08_mcp_service_detail.png)

### 怎么用

1. 点击服务标签查看详情
2. 点击 **连接** 建立连接
3. 连接成功后，状态徽章变绿
4. 现在聊天时，AI 就可以使用这个工具了
5. 用完后点击 **断开**

> **小提示**：可以同时连接多个服务！

## 3. 角色 — 设定 AI 的性格

角色就是系统提示词，告诉 AI 应该怎么表现。比如选一个"Blender 专家"角色，AI 就会像 3D 建模专家一样回答你。

![角色管理页面](img/09_roles_overview.png)

### 内置分类

| 分类 | 用途 |
|---|---|
| **Custom（自定义）** | 你自己创建的角色 |
| **System（系统）** | 内置的默认角色 |
| **Blender** | 适合 3D 建模任务的角色 |
| **GIMP** | 适合图片编辑任务的角色 |

### 怎么创建角色

1. 点击 **+ New Role**
2. 填写 **标题**（如"Blender 专家"）
3. 选择 **分类**
4. 编写 **内容** —— 描述 AI 应该怎么表现
5. 点击 **保存**

![新建角色对话框](img/10_roles_new_dialog.png)

### 怎么使用角色

在聊天页面底部，从角色下拉列表中选择一个角色。对话中 AI 就会按照角色指令来回答。

### 编辑角色

直接在表格中编辑——点击任意行的 **编辑** 按钮，或直接点击标题/内容单元格。

> **小提示**：使用分类标签（全部/自定义/系统/Blender/GIMP）来过滤列表。

## 4. 提示词 — 保存常用消息

把经常发送给 AI 的话保存为模板，一键复用。

![用户提示词页面](img/11_prompts_overview.png)

### 怎么用

1. 点击 **+ New Prompt**
2. 填写 **标题** 和 **内容**
3. 点击 **保存**
4. 想用的时候，点击模板上的 **复制** 按钮
5. 到聊天页面粘贴发送即可

## 5. 模型 — 添加 AI 模型

你可以添加来自不同供应商的模型。系统自带了很多热门模型的预设。

![模型设置页面](img/12_models_overview.png)

### 支持的供应商

DeepSeek、OpenAI（GPT-4o）、Anthropic（Claude）、Google（Gemini）、Moonshot、阿里云（通义千问）、百度（ERNIE）、腾讯（混元）、Groq、Together AI、xAI、OpenRouter、Ollama（本地）等等。

### 怎么添加模型

1. 点击 **+ Add Model**
2. 点击 **Quick fill** 按钮自动填充热门模型
3. 或者手动填写：
   - **供应商**：谁提供的模型（如 DeepSeek）
   - **名称**：你容易识别的名字
   - **模型**：模型 ID（如 deepseek-chat）
   - **接口地址（Base URL）**：API 地址
   - **API Key**：你的密钥
   - **代理地址**：（可选）如果需要代理
4. 点击 **创建**

![添加模型](img/13_models_add_form.png)

### 其他操作

| 按钮 | 作用 |
|---|---|
| **Set Active** | 把这个模型设为聊天的默认模型 |
| **Test** | 测试连接是否正常 |
| **Edit** | 修改设置 |
| **Delete** | 删除这个模型 |

## 6. 设置 — 系统配置

### 通知设置

配置 Webhook（如飞书机器人），当 AI 完成任务时自动通知你。

![通知设置](img/14_settings_notify.png)

- **Webhook URL**：通知机器人的地址
- **关键词**：AI 回复中包含这些词时就发通知
  - 留空则每次任务完成都通知

### 其他设置

![其他设置](img/15_settings_other.png)

- **Blender 工作目录**：Blender 生成的文件保存的位置

## 快速开始

1. **添加模型**（模型页面）—— 点击预设、填入 API Key、点击创建、再点 Set Active
2. **连接工具**（AI 桥接页面）—— 点击 Blender 或 GIMP 的连接按钮
3. **开始聊天**（聊天页面）—— 点击 + New、输入消息、按发送
4. **设定角色**（可选）—— 从下拉列表选一个角色来引导 AI
